# Cafe Street - Landing Page

This is a landing page for an online ordering website, built with Vite, React, and styled using Tailwind CSS. The design is inspired by a free Figma template.

## Installation

### Prerequisites

- **Node.js** and **npm** installed on your machine. If not, download it from [Node.js Official Site](https://nodejs.org/).

### Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/rockmel03/cafe-street.git
   ```

2. Navigate into the project directory:

```bash
cd your-repo-name
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

This will start the Vite development server. You can view the app by navigating to `http://localhost:3000` in your browser.
